## Инструкция по запуску проекта

Для запуска проекта выполните следующие действия:

1. Установите необходимые зависимости с помощью команды:
    ```bash
    pip install -r requirements.txt
    ```
2. Запустите приложение, выполнив:
    ```bash
    python main.py
    ```
